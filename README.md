# Gestoría Financiera JS

## Descripción
Proyecto de una aplicación web sencilla para la gestión financiera, dividida en tres módulos principales: Clientes, Carteras y Stocks. Cada módulo permite dar de alta, editar, borrar, buscar y listar registros, facilitando la administración de información financiera.

## Distribución del proyecto
- Apartado Clientes: Guillermo Andújar Martínez
- Apartado Carteras: Guillermo Jiménez Martínez
- Apartado Stocks: Alan Fernández

## Funcionalidades principales
- Gestión completa de Clientes, Carteras y Stocks.
- Operaciones CRUD (Crear, Leer, Actualizar, Borrar).
- Buscador para filtrar resultados en cada módulo.
- Listado completo y por parámetros.
- Navegación separada para cada módulo.

## Tecnologías utilizadas
- JavaScript para la interfaz y lógica frontend.
- PHP para la gestión backend y conexión con la base de datos.
- Base de datos MySQL.
- HTML para estructuración.
